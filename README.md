# Pigie Finder

A community platform where college students discover, review, and directly contact trusted local vendors and service providers near their campus.

**Status:** Phase 0 (Planning & Requirements) complete. See `build.md` for the full phased build plan.

**Tech stack:** Node.js, Express, EJS, CSS, JavaScript (vanilla), SQL (MySQL/PostgreSQL), Google Maps API

## Repo layout

```
pigie-finder/
├── build.md              # Full phased build plan
├── docs/
│   ├── phase0-scope.md   # Locked feature scope, roles, and decisions
│   └── wireframes.html   # Low-fidelity wireframes for the 6 core screens
├── PHASE_CHECKLIST.md    # One checklist per build phase (mirror as GitHub issues)
└── .gitignore
```

Application code (`routes/`, `controllers/`, `models/`, `views/`, etc.) is scaffolded starting in Phase 1.
